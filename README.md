# 幹話產生器
透過 Express 建立的幹話產生器
![](/rubbishTalkGenerator.png)
## 功能描述
- 三個職業可供選擇
- 每個職業都可以產生不同的幹話
### 環境建置與需求
* Node.js: v10.15.0
* npm: 6.4.1
* Nodemon: 2.0.4
* express: 4.17.1
* express-handlebars: 5.1.0
* body-parser: 1.19.0
* Bootstrap: v4.2.1
* jQuery: v3.3.1
* popper 1.12.5 
### 安裝與執行步驟
開啟終端機 (Terminal)，clone 此專案至本機(Local)電腦
```
$ git clone https://github.com/Anderson-Chiang/rubbish-talk.git
```

進入專案資料夾
```
$ cd rubbish-talk
```

安裝 npm 套件
```
$ npm i
```

安裝 nodemon 套件，並啟動伺服器，執行 app.js 檔案
```
$ npm run dev
```

當終端機出現以下字樣，表示伺服器已連結成功
```
The server is listening on http://localhost:3000
```

在瀏覽器輸入網址，並進入網站
```
http://localhost:3000
```
